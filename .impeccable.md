# L'Agence Grey - Design Context

This file is the design context for Impeccable skills.
Generated for L'Agence Grey web agency project.

## Target Audience
Studios de yoga, pole dance, sport et bien-être en France.
Cibles : propriétaires de studios (30-50 ans), pas très tech,
qui veulent un site professionnel sans se prendre la tête.

## Brand Personality
- **Dark & premium** mais accessible (pas froide)
- **Purple accents** sur fond sombre (#1a1a1a base)
- Moderne, clean, pas de fioritures
- Touche humaine malgré l'aspect tech
- Confiance et professionnalisme

## Design System
- **Colors**: Dark base #1a1a1a, purple accents, OKLCH preferred for dark mode
- **Fonts**: Poppins (headings) + Nunito (body) - DO NOT use Inter, Roboto, Open Sans
- **Spacing**: 4px grid base
- **Border radius**: Moderate (not extreme)
- **Motion**: Subtle, purposeful, no bounce easing

## Use Cases
- Landing pages pour studios de sport
- Pages de services / tarifs
- Portfolio de réalisations
- Formulaire de contact / prise de RDV
- Pages légales (mentions, CGV)

## Tone
Copy en français, tutoiement.
Clair, direct, pas de jargon marketing.
CTA orientés action concrète ("Demander un devis", "Voir nos réalisations").

## Anti-Patterns Specific to This Project
- NEVER use pure black (#000000) - use #1a1a1a or tinted dark
- NO purple gradients everywhere (one accent color, not a rainbow)
- NO cards-in-cards nesting
- NO Inter font (brand uses Poppins/Nunito)
- NO gray text on colored backgrounds (accessibility)
- NO bounce/elastic easing on animations
- Keep text readable on dark backgrounds (increase line-height slightly)
